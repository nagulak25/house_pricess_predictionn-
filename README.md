# 🏠 House Price Prediction Web App

A machine learning web application that predicts house prices based on user input features. Built using Flask and Scikit-learn with a clean interactive UI.

---

## 🚀 Features

- 📊 Automatic dataset processing
- 🤖 ML model training (Random Forest & Ridge)
- 🎯 Real-time price prediction
- 📉 Confidence interval output
- 💾 Model saving & reuse
- 🌐 Simple web interface

---

## 🛠️ Tech Stack

- Python (Flask)
- Pandas, NumPy
- Scikit-learn
- HTML, Bootstrap, JavaScript
